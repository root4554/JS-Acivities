$("*") The * selector selects all elements in the document, including html, head and body.
$(this) the this keyword refers to the current element in the selection.
$("p.intro") The p.intro selector selects all paragraphs with the specific class (.intro).
$("p:first") The p.first selects first paragraphs.
$("p:eq(2)") The p:eq(2) selects 3th, third paragraphs.
$("ul li:first") The ul li:first selects first li in ul.
$("ul li:first-child") The ul li:first-child selects first li in ul.
$("[href]") The [href] selector selects all elements with an href attribute.
$("a[target='_blank']") The a[target='_blank'] selector selects all elements with a target attribute value equal to "_blank".
$("a[target!='_blank']") The a[target!='_blank'] selector selects all elements with a target attribute value NOT equal to "_blank".
$(":button") The :button selector selects all button elements and input elements of type="button".
$("tr:even") The tr:even selector selects all even tr elements.
$("tr:odd") The tr:odd selector selects all odd tr elements.
